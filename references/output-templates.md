# Output Templates Reference

This document contains detailed templates for all output document types generated by the Meeting Insights Analyzer skill.

## Study Guide Template

```markdown
# [Topic] Study Guide
## Generated from: [Meeting Title/Date]

### Learning Objectives
After reviewing this material, you should be able to:
- [Objective 1]
- [Objective 2]
- [Objective 3]

### Key Concepts

#### [Concept 1 Name]
**Definition:** [Clear explanation]
**Why It Matters:** [Relevance to role]
**Example:** [Practical example from transcript]

#### [Concept 2 Name]
[Continue pattern...]

### Important Procedures
1. [Procedure name]
   - Step 1: [Action]
   - Step 2: [Action]
   - Key tip: [Insight from trainer]

### Key Contacts
| Role | Name | Contact | When to Reach Out |
|------|------|---------|-------------------|
| [Role] | [Name] | [Email/Slack] | [Situation] |

### Review Questions
1. [Question to test understanding]
2. [Question to test understanding]
3. [Question to test understanding]

### Additional Resources
- [Resource 1]: [Location/Link]
- [Resource 2]: [Location/Link]
```

## Quick Reference Sheet Template

```markdown
# Quick Reference: [Topic]
*One-page summary for daily use*

## Essential Information
| Item | Details |
|------|----------|
| [Key Item] | [Critical Info] |

## Common Tasks
**[Task Name]:** [Brief steps]

## Key Contacts
- **[Role]:** [Name] - [Contact]

## Important Links
- [System Name]: [URL or location]

## Remember
- [Critical reminder 1]
- [Critical reminder 2]
```
